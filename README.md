#  Personalized Webpage Project

A sleek, interactive web application that captures user data, performs real-time age calculations, and persists information using browser storage.

##  Features
- **Data Persistence**: Uses `localStorage` to remember your name even after a refresh.
- **Dynamic Logic**: Calculates your age in months and checks for age-restricted content.
- **Glassmorphism UI**: A modern, frosted-glass interface built with pure CSS.
- **Responsive Design**: Works perfectly on mobile and desktop.

##  Tech Stack
- **HTML5** (Structure)
- **CSS3** (Glassmorphism & Layout)
- **JavaScript** (ES6+ Logic)
